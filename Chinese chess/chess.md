# 中国象棋游戏说明

## 简介

这是一个用Python编写的中国象棋游戏命令行版本。游戏实现了基本的棋盘布局、棋子移动规则和轮流对弈功能。

## 文件结构

- `chess.py`: 主游戏文件，包含所有游戏逻辑。

## 游戏规则

### 棋盘
- 10行 x 9列的棋盘
- 红方在底部（第9行），黑方在顶部（第0行）

### 棋子
- **帅(K)**: 只能在九宫格内移动，一次移动一格（上下左右）
- **士(A)**: 只能在九宫格内斜线移动
- **相(B)**: 田字形移动，不能过河，不能被塞象眼
- **马(N)**: 日字形移动，不能被别马腿
- **车(R)**: 直线移动，可以吃子
- **炮(C)**: 隔山打牛，需要一个棋子作为炮架
- **兵(P)**: 过河前只能前进，过河后可以左右移动

### 游戏流程
1. 红方先走
2. 玩家轮流输入起始位置和目标位置
3. 程序验证移动是否合法
4. 游戏继续直到一方获胜（当前版本未实现将死检查）

## 使用方法

### 运行游戏
```bash
python chess.py
```

### 游戏操作
1. 程序显示当前棋盘
2. 输入提示："输入起始位置 (行 列): "
3. 输入起始位置的行号和列号，用空格分隔
4. 输入提示："输入目标位置 (行 列): "
5. 输入目标位置的行号和列号，用空格分隔
6. 程序验证并执行移动，或提示"无效移动"

### 示例输入
```
输入起始位置 (行 列): 9 1
输入目标位置 (行 列): 7 2
```

## 代码结构

### 类和方法

#### ChessPiece 类
- `__init__(self, piece_type, color)`: 初始化棋子
- `__str__(self)`: 返回棋子中文名称

#### ChineseChess 类
- `__init__(self)`: 初始化游戏
- `setup_board(self)`: 设置初始棋盘
- `print_board(self)`: 打印棋盘
- `is_valid_move(self, start_row, start_col, end_row, end_col)`: 验证移动是否合法
- `make_move(self, start_row, start_col, end_row, end_col)`: 执行移动
- `is_checkmate(self)`: 检查是否将死（当前返回False）
- `play(self)`: 主游戏循环

### 全局变量
- `PIECES`: 棋子符号到中文名称的映射
- `INITIAL_BOARD`: 初始棋盘布局

## 注意事项

1. **简化实现**: 这个版本的移动验证是简化的，没有实现所有复杂的规则，如：
   - 将帅对峙
   - 长捉
   - 某些特殊情况的移动限制

2. **输入验证**: 程序对输入有基本错误处理，但建议输入正确的数字。

3. **游戏结束**: 当前版本没有实现游戏结束条件检测。

4. **扩展建议**: 可以添加图形界面、AI对手、完整规则验证等功能。
